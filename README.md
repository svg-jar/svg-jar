# SVG Jar

SVG Jar is a collection of tools for working with SVG files on the web.

## Packages

- [Core](./packages/core) - A plugin for Vite, Webpack, and Rollup to import SVG files for Ember, React, Preact, Vue and Svelte.
- [codemod](./packages/codemod) - A codemod to upgrade from [ember-svg-jar](https://github.com/evoactivity/ember-svg-jar)
- [browser](./packages/browser) - A UI for browsing your applications SVG files
